# 何时使用
用户要求“回复 YouTube 评论”等任务时，用于自动回复 YouTube 评论，保持互动性。

# 大致步骤：
1. 使用用户指定浏览器，打开 https://studio.youtube.com/ （全程仅在 `studio.youtube.com` 域内操作，不打开其他网站或外链。）
2. 定位评论区（通常视频的入口在用户的主页（边栏中的内容，然后再点击视频）中能找到，最好不要在搜索框进行搜索）
3. 点击对应的视频
4. 点击边栏的评论。
5. 你应该能看到顶部有几个评论筛选按键，你可以自行审查，根据用户需求，对按键进行点击。
6. 分析页面结构，判断按钮与评论的对应关系，根据用户的需求，进行评论回复。

# 页面排布逻辑
- 有两种评论：主评论 & 子评论（二级评论 / 回复），请严格遵循用户指令，对任务进行操作。
